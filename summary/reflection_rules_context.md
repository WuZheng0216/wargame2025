# 规则与得分摘要（供反思智能体使用）

## 1. 目标函数
- 本版规则不再给红蓝方设置额外任务完成分，核心目标是以最小代价最大化对手损失。
- 总分公式：`总分 = 毁伤得分 - 经济惩罚 + 对方违规扣分`
- 反思时优先判断：是否打出了高价值目标毁伤、是否控制了经济消耗、是否触发了违规扣分。

## 2. 红方关键能力与约束
- 红方主要得分对象是蓝方 `旗舰 / 驱逐舰 / 巡洋舰`，按损伤比例计分。
- 红方机动发射车：成本 `4`，抗毁伤 `150`，速度 `48m/s`。
- 红方高成本攻击弹：成本 `8`，基础伤害 `50`，速度 `1200m/s`，射程 `400km`。
- 红方低成本攻击弹：成本 `3`，基础伤害 `15`，速度 `800m/s`，射程 `350km`。
- 机动发射车发射高成本/低成本攻击弹的冷却时间均为 `60s`。
- 发射车挂载的高成本弹和低成本弹总量受总成本约束；简化为可补弹，不单独限制挂载枚数。
- 红方引导快艇：成本 `2`，抗毁伤 `80`，速度 `70m/s`，对空/对海探测半径均为 `50km`。
- 红方侦察无人机：成本 `5`，抗毁伤 `10`，速度 `590m/s`，探测半径 `60km`。
- 高低成本攻击弹带末制导，末制导范围 `7km`；飞行中可接收己方共享态势信息。
- 红方命中率：
  - 高成本攻击弹打击蓝方舰船：无引导 `70%`，有引导 `90%`
  - 低成本攻击弹打击蓝方舰船：无引导 `40%`，有引导 `60%`
  - 引导快艇可额外提升高低成本攻击弹命中率 `20%`
- 红方发射车发弹后会从隐蔽转为暴露；隐蔽状态下不会被蓝方卫星发现，但仍可能被其他手段探测。

## 3. 蓝方关键能力与约束
- 蓝方旗舰：成本 `24`，抗毁伤 `200`，速度 `36m/s`，舰对地巡航导弹最多 `8` 枚，冷却 `120s`。
- 蓝方驱逐舰：成本 `16`，抗毁伤 `100`，速度 `54m/s`。
  - 近程拦截弹：`60km`
  - 远程拦截弹：`100km`
  - 拦截弹总量最多 `15`
  - 舰对地巡航导弹最多 `16`
- 蓝方巡洋舰：速度 `45m/s`。
  - 近程拦截弹：`60km`
  - 远程拦截弹：`100km`
  - 拦截弹总量最多 `30`
  - 舰对地巡航导弹最多 `12`
- 蓝方舰载机：成本 `16`，抗毁伤 `20`，速度 `590m/s`。
  - 对海雷达 `50km`，对空雷达 `50km`
  - 空空导弹 `50km`，空面导弹 `50km`
  - 发射冷却 `0s`
  - 每架最多挂 `4` 枚空空导弹，或最多挂 `2` 枚空面导弹，二选一
- 蓝方电子战设备仅可部署在 `驱逐舰 / 旗舰 / 舰载机` 上：
  - 干扰范围 `50km`
  - 每次开启持续 `5分钟`
  - 冷却 `10分钟`
  - 主要压制红方无人机、导弹、快艇探测设备
  - 会使红方无人机探测范围和导弹命中率下降为原来的 `80%`

## 4. 探测、机动、拦截与毁伤规则
- 探测有效必须同时满足：`目标在范围内` 且 `满足识别概率要求`。
- 卫星默认对核心区域全覆盖侦察，但双方卫星都无法探测飞行中的导弹。
- 蓝方舰载雷达探测半径 `50km`，主要针对各类飞行器；对海瞭望 `15km`。
- 红方侦察无人机探测半径 `60km`，引导快艇探测半径 `50km`。
- 舰船雷达静默时，进攻方通常只能依赖无人机等侦察手段获取具体位置；舰船开雷达会暴露给红方卫星。
- 机动受域限制：发射车仅能陆上机动；舰船/快艇仅海上机动；导弹、无人机、舰载机在空域机动。
- 指令更新规则：同一帧同一装备只执行最后一条机动指令；跨帧按最新指令覆盖。
- 拦截有效必须同时满足：`目标在拦截范围内` 且 `满足拦截概率要求`；交汇距离小于 `50m`。
- 拦截概率：
  - 远程拦截弹 `70%`
  - 近程拦截弹 `40%`
- 毁伤采用线性累积；累计超过 `100%` 视为最终摧毁。
- 其他命中率：
  - 空空导弹 `40%`
  - 空面导弹 `60%`
  - 舰对地巡航导弹 `60%`

## 5. 得分与惩罚规则
- 红方对蓝方目标毁伤得分：
  - `旗舰 150分/艘`
  - `驱逐舰 50分/艘`
  - `巡洋舰 80分/艘`
  - 均按损伤比例累计
- 蓝方对红方目标毁伤/拦截得分：
  - `机动发射车 60分/辆`
  - `引导快艇 20分/艘`
  - `高成本攻击飞行器 8分/枚`
  - `低成本攻击飞行器 4分/枚`
  - `侦察无人机 1分/架`
- 红方经济惩罚：
  - 总经济消耗超过蓝方 `50%` 以上才开始扣分
  - 每超 `10%` 扣 `10分`
  - 上限 `100分`
  - 若目标完成，惩罚减半
- 蓝方经济惩罚：
  - 总经济消耗超过红方 `20%` 以上才开始扣分
  - 每超 `10%` 扣 `10分`
  - 上限 `100分`
- 违规扣分：
  - 商船保护：武器或舰艇威胁商船 `500m` 范围内，扣 `50分/次`


## 6. 反思时应重点判断
- 是否围绕高价值目标得分，而不是只追求动作数量。
- 是否正确利用了 `侦察无人机 / 引导快艇 / 发射车` 的分工关系。
- 是否在高低成本弹之间做了符合成本效益的选择。
- 是否在蓝方拦截、雷达、干扰条件下仍保持了有效突防窗口。
- 是否因为暴露、静默、探测链断裂或冷却约束而错失火力窗口。
- 是否触发了商船保护、部署分散等可避免的违规扣分。
- 生成经验时优先输出可执行教训，例如：目标选择、侦察节奏、引导时机、齐射规模、弹药经济性、规避干扰/拦截的方法。
